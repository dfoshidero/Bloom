Windows:
1. Install Git https://git-scm.com/download/win
2. Open Command Prompt and navigate to the folder where you want to put your local repository
    (note that the repository's root folder will be INSIDE that folder)
3. Create the local repository with the command: git clone https://github.com/dfoshidero/Bloom
4. Enter that repository and enter the command: npm install
5. Then build and run the app with the command: npx expo start
6. Download Expo Go to your phone and scan the QR code
